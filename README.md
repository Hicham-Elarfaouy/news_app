#  News App ✨


News is open-source  app for Android & ios. It is built with Flutter Framework.

Star ⭐ the repo if you like it.

<p><img src="snapshots/Home.png" height="500" /></p>



## Features

 - Daily News 
 - Categories Business & Sport & Science
 - Multi Country
 - Dark Theme
 - Search News
